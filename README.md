Sum of Two Numbers — Django CRUD App
A simple Django web application that performs addition of two numbers and stores every operation in a SQLite database. Supports full Create, Read, Update, Delete (CRUD) on all past operations.

🚀 Features

Add two numbers and instantly see the result
All operations saved to a database (persists after server restart)
View complete history of all past operations in a table
Edit any previous operation and auto-recalculate the result
Delete any operation from the table
Built with Bootstrap 5 for a clean, responsive UI
